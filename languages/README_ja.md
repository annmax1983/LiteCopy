# LiteCopy
[English](../README.md) | [中文](README_zh.md) | [Español](README_es.md) | [Deutsch](README_de.md) | 日本語 | [Français](README_fr.md)

あらゆるウェブサイトのテキスト選択、右クリックメニュー、コピー/貼り付けショートカットを復元する軽量ブラウザ拡張機能。

> Chromium · Manifest V3 · 最小権限 · ローカル

---

## なぜ LiteCopy？

ウェブサイトからテキストをコピーしようとして、選択できなかったり、右クリックがブロックされたり、Ctrl+Cが効かなかったことはありませんか？LiteCopy はワンクリックで全て解決します。

| 利点 | 詳細 |
|------|------|
| 🔓 **ワンクリックで復元** | テキスト選択を瞬時に復元、ページ更新不要 |
| 🔒 **最小権限** | `activeTab` + `scripting` のみ |
| ⚡ **軽量** | 15KB未満、パッケージ化で10KB。フレームワークなし |
| 🌍 **6言語** | 英語、中国語、スペイン語、ドイツ語、日本語、フランス語 |
| 🚫 **ゼロトラッキング** | 分析なし、ネットワークリクエストなし |

---

## 機能

| 機能 | 説明 |
|------|------|
| 🔓 **テキスト選択スタイルを復元** | `user-select:none` を上書き |
| 🖱️ **右クリックメニューを復元** | ブロックされたコンテキストメニューを復元 |
| ⌨️ **キーボードショートカットを復元** | Ctrl+C、Ctrl+V、Ctrl+A を復元 |
| 🛡️ **オーバーレイを除去** | テキスト選択をブロックする透明レイヤーを無効化 |
| 🔄 **ON/OFF切替** | アイコンクリックで有効/無効を切替 |

## インストール

1. ブラウザの拡張機能ページを開く
2. **デベロッパーモード**を有効にする
3. **パッケージ化されていない拡張機能を読み込む**をクリックし、`lite-copy` フォルダを選択

---

## 使い方

1. コピー制限のあるウェブサイトにアクセス
2. **LiteCopy** アイコンをクリック
3. テキスト選択が瞬時に復元
4. もう一度クリックで無効化（ページが再読み込み）
