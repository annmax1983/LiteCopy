# LiteCopy
[English](../README.md) | 中文 | [Español](README_es.md) | [Deutsch](README_de.md) | [日本語](README_ja.md) | [Français](README_fr.md)

一款轻量级浏览器扩展，一键恢复网页文本选中、右键菜单和复制粘贴功能。

> Chromium 内核 · Manifest V3 · 最小权限 · 100% 本地

---

## 为什么选择 LiteCopy？

遇到过网页不让复制、不让选中文字、右键被拦截的情况吗？LiteCopy 一键搞定。

| 优势 | 说明 |
|------|------|
| 🔓 **一键启用** | 瞬间恢复文本选中功能，无需刷新页面 |
| 🔒 **最小权限** | 仅 `activeTab` + `scripting`，不申请多余权限 |
| ⚡ **极简轻量** | 源码不到 15KB，打包仅 10KB，无框架无依赖 |
| 🌍 **6 种语言** | 英文、中文、西班牙文、德文、日文、法文 |
| 🚫 **零追踪** | 无分析、无网络请求、无数据上传 |
| 🎯 **简单开关** | 点击图标切换开启/关闭，状态一目了然 |

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 🔓 **恢复文本选中样式** | 覆盖 `user-select:none` 和 `-webkit-user-select:none` |
| 🖱️ **恢复右键菜单** | 恢复被拦截的右键菜单 |
| ⌨️ **恢复复制粘贴快捷键** | 恢复 Ctrl+C、Ctrl+V、Ctrl+A 等复制粘贴快捷键 |
| 🛡️ **清除遮罩层** | 禁用阻止文本选中的透明覆盖层 |
| 🔄 **开关切换** | 点击工具栏图标启用/禁用，禁用时自动刷新页面 |
| 💬 **轻提示** | 自动消失的提示，告知激活状态 |
| 📋 **复制页面信息** | Popup 按钮：复制页面标题、链接或两者 |

## 安装方法

1. 打开浏览器扩展页面：
   - **Chrome**：`chrome://extensions/`
   - **Edge**：`edge://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**，选择 `lite-copy` 文件夹
4. 工具栏中出现 LiteCopy 图标

---

## 使用方法

1. 访问任何限制复制或选中的网页
2. 点击工具栏中的 **LiteCopy** 图标
3. 文本选中功能瞬间恢复
4. 再次点击图标关闭（页面自动刷新恢复原状）

**复制页面信息：**
- 点击图标 → 打开 Popup
- 使用按钮复制页面标题、链接或两者
